# Документация проекта

Эта директория содержит дополнительную документацию проекта "Программная инженерия".

## Структура документации

Основная документация проекта находится в [Wiki](../../wiki) репозитория. Эта директория используется для хранения дополнительных материалов и файлов, которые не подходят для Wiki формата.

## Содержание

В будущем здесь будут размещены:

- Технические спецификации
- API документация
- Диаграммы архитектуры
- Руководства пользователя
- Отчёты и аналитика

## Использование

Файлы в этой директории могут быть в различных форматах:
- Markdown (`.md`) для текстовой документации
- Диаграммы (`.png`, `.svg`, `.drawio`)
- PDF документы для формальной документации
- Другие форматы по необходимости

## Связанные ресурсы

- [Wiki](../../wiki) - основная документация
- [README.md](../README.md) - описание проекта
- [CONTRIBUTING.md](../CONTRIBUTING.md) - руководство по внесению изменений
