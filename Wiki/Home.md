# Добро пожаловать в Wiki проекта "Программная инженерия"

Приветствуем вас в Wiki-документации учебного проекта по курсу "Программная инженерия"!

## О проекте

Этот проект создан для практического освоения современных методов и инструментов программной инженерии. Мы используем GitHub в качестве платформы для изучения полного цикла разработки программного обеспечения, включая:

- 📝 Документирование проектов
- 🔄 Системы контроля версий (Git)
- 🚀 DevOps практики и CI/CD
- 📊 Управление проектами и задачами
- 👥 Командная работа и code review

## Зачем используется Wiki?

Wiki репозитория служит централизованным местом для хранения и управления документацией проекта. Это позволяет:

### Преимущества использования Wiki

- **Простота редактирования**: Wiki использует Markdown, который легко читать и писать
- **Версионирование**: все изменения в Wiki отслеживаются через Git
- **Доступность**: документация всегда доступна рядом с кодом
- **Коллаборация**: несколько человек могут работать над документацией одновременно
- **Структурированность**: логическая организация информации по страницам и разделам

### Что хранится в Wiki

1. **Документация по архитектуре**: описание структуры и компонентов системы
2. **Руководства для разработчиков**: инструкции по настройке окружения и процессам разработки
3. **Планирование**: календарь событий, планы спринтов и релизов
4. **Справочная информация**: API документация, конфигурации, troubleshooting
5. **Учебные материалы**: примеры, туториалы, best practices

Используйте боковую панель навигации (Sidebar) слева для быстрого перехода между страницами Wiki.

## Как внести изменения

Подробные инструкции по работе с Wiki и внесению изменений вы найдете в файле [CONTRIBUTING.md](https://github.com/sl4sh73r/Software-Engineering-/blob/main/CONTRIBUTING.md) основного репозитория.

Краткая инструкция:
1. Нажмите кнопку **"Edit"** на любой странице для редактирования через веб-интерфейс
2. Или клонируйте Wiki репозиторий локально для работы в вашем редакторе
