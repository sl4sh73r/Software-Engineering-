# События Января

Планируемые мероприятия и задачи на январь в рамках курса "Программная инженерия".

## Обзор месяца

Январь - месяц начала курса и настройки базовой инфраструктуры проекта. Основной фокус на освоение инструментов и создание фундамента для дальнейшей работы.

## Недели января

### Неделя 1 (1-7 января)

**Праздничные дни**: Новогодние каникулы
- Подготовка к началу курса
- Ознакомление с материалами курса

### Неделя 2 (8-14 января)

**Тема**: Введение в программную инженерию и GitHub

**Мероприятия**:
- 🎓 **Лекция**: Введение в курс "Программная инженерия"
  - Дата: 9 января, 10:00
  - Темы: Обзор курса, цели, результаты обучения
  
- 🎓 **Лекция**: Системы контроля версий и Git
  - Дата: 11 января, 10:00
  - Темы: Основы Git, ветвление, слияние

**Задачи**:
- [ ] Создать аккаунт на GitHub
- [ ] Настроить локальное окружение разработки
- [ ] Клонировать учебный репозиторий
- [ ] Изучить базовые команды Git

**Дедлайны**:
- Настройка окружения: до 14 января

### Неделя 3 (15-21 января)

**Тема**: Работа с GitHub и документирование

**Мероприятия**:
- 🎓 **Лекция**: GitHub как платформа для совместной разработки
  - Дата: 16 января, 10:00
  - Темы: Issues, Pull Requests, Code Review
  
- 💻 **Практикум**: Работа с Wiki и документацией
  - Дата: 18 января, 14:00
  - Практика создания и редактирования Wiki страниц

**Задачи**:
- [ ] Создать первый Issue
- [ ] Сделать первый Pull Request
- [ ] Добавить страницу в Wiki проекта
- [ ] Провести Code Review для коллеги

**Дедлайны**:
- Первый PR: до 21 января

### Неделя 4 (22-28 января)

**Тема**: Введение в DevOps и CI/CD

**Мероприятия**:
- 🎓 **Лекция**: Основы DevOps
  - Дата: 23 января, 10:00
  - Темы: Принципы DevOps, CI/CD, автоматизация
  
- 🎓 **Лекция**: GitHub Actions - основы
  - Дата: 25 января, 10:00
  - Темы: Workflows, Jobs, Steps, Actions

**Задачи**:
- [ ] Изучить существующий CI/CD pipeline проекта
- [ ] Создать простой GitHub Actions workflow
- [ ] Запустить первую автоматическую сборку
- [ ] Проанализировать результаты CI/CD

**Дедлайны**:
- Создание workflow: до 28 января

### Неделя 5 (29-31 января)

**Тема**: Подведение итогов января

**Мероприятия**:
- 📊 **Ретроспектива**: Итоги января
  - Дата: 30 января, 15:00
  - Обсуждение успехов и сложностей

**Задачи**:
- [ ] Завершить все задачи января
- [ ] Подготовить отчёт о проделанной работе
- [ ] Внести предложения по улучшению процесса

## Ключевые цели января

### Технические навыки
- ✅ Освоить базовые операции с Git

- ✅ Понять основы CI/CD
- ✅ Создать первый GitHub Actions workflow

### Командная работа
- ✅ Познакомиться с участниками курса
- ✅ Провести первые code reviews
- ✅ Научиться документировать работу

### Инфраструктура проекта
- ✅ Настроить репозиторий
- ✅ Создать структуру Wiki
- ✅ Настроить базовый CI/CD pipeline
